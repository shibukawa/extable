# Button & Link Cells

Buttons emit action payloads and links navigate to URLs. Both are readonly and non-editable.

## What You're Seeing

- **Button Cells** - Trigger actions via click or Space  
- **Link Cells** - Navigate to URLs with a label or URL text  
- **Readonly by Default** - Actions are interactive but not editable  
- **Event Payloads** - Button actions show a payload in the log
